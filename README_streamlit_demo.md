# Streamlit Demo Dashboard

## What this app does

This is a judge-facing Streamlit demo for the upgraded nonprofit resilience system. It is a lightweight decision-support layer on top of existing rerun outputs, not a production app and not a new modeling pipeline.

The app stays anchored to the retained technical story:

- Backbone: upgraded logistic regression
- Label: `peer_relative_composite`
- Retained pattern: peer-anchored features without direct current-year margin or reserve levels in the retained score

## Tabs in the demo

1. `Risk Score Lookup`
   Shows baseline distress probability, risk bucket, driver summary, and threshold flags for one selected nonprofit.
2. `Peer Benchmarking`
   Shows reserve, margin, leverage, and concentration position relative to peers.
3. `Shock Simulator`
   Shows deterministic downside stress tests under donation, program revenue, grant / government, and expense inflation shocks.
4. `High-Impact Discovery`
   Shows intervention-ready lists for fragile but investable organizations, resilient outperformers, and a shock-sensitive watchlist.
5. `Portfolio Rankings`
   Shows macro-level rankings across the full filtered portfolio so judges can scan strongest, riskiest, most shock-sensitive, and most intervention-ready organizations before drilling back into the org-level tabs.

## Portfolio Rankings behavior

The new `Portfolio Rankings` tab is a macro control room built from existing downstream artifacts.

It supports ranking modes for:

- highest baseline risk
- lowest baseline risk / most resilient
- worst peer-relative reserve weakness
- worst leverage pressure
- highest shock sensitivity
- largest bucket jump under scenario stress
- fragile but investable
- resilient outperformers
- shock-sensitive watchlist
- strongest reserve position
- strongest peer-relative strength
- highest concentration risk
- biggest donation-shock sensitivity
- biggest expense-inflation sensitivity

It also supports:

- macro filters for peer group, size band, funding model, state, sector, risk bucket, shortlist category, and scenario type
- top-N and sort-direction controls
- summary tiles that react to the filtered slice
- one selected-row detail panel
- sync back to the shared organization selector through `Use selected org in main dashboard`

Sync behavior:

- The selected organization in the sidebar still drives Tabs 1 through 4.
- In `Portfolio Rankings`, choose a row in the detail selector and click `Use selected org in main dashboard`.
- That updates the shared session-state organization and immediately reruns the app so the original 4 tabs stay aligned.

## Run locally

1. Install dependencies:

```powershell
python -m pip install -r requirements_streamlit_demo.txt
```

2. Rebuild the exact upgraded org-level shock artifact if needed:

```powershell
python scripts\build_streamlit_demo_artifacts.py --force
```

3. Launch the dashboard:

```powershell
python -m streamlit run app.py
```

## What is exact vs approximate

Exact / reused from upgraded reruns:

- baseline risk lookup
- peer benchmark cards
- shortlists
- threshold evidence
- upgraded org-level shock rows generated by `scripts/build_streamlit_demo_artifacts.py`
- portfolio ranking logic that reuses those same baseline, peer, shock, and shortlist outputs

Approximate / derived in the demo layer:

- top-driver text for individual rows when native row-level explanations are not stored
- warning-chip phrasing built from retained threshold outputs
- plain-English narrative summaries
- portfolio rationale text when only benchmark or shock summaries are available
- donor vs grant / government exposure interpretation, which inherits the repo's filing-structure approximation

Fallback behavior:

- If the upgraded org-level shock artifact is missing, the simulator falls back to the packaged org-level simulator if available.
- If no org-level shock rows are available, the simulator still renders a summary-level scenario view.
- If raw latest-year structural fields are missing, the app still runs on benchmark cards, but lookup and portfolio detail become thinner.
- If a portfolio ranking mode cannot be supported from the current artifact set, the tab shows a note instead of crashing.
